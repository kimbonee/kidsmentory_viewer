# 🎬 Vimeo 영상 뷰어

[Vimeo 사용자 user104908181](https://vimeo.com/user104908181)의 모든 영상을 썸네일 목록 형태로 탐색하고, 클릭 시 상세 페이지를 새창으로 열어주는 웹 애플리케이션입니다.

## ✨ 주요 기능

- **📱 반응형 디자인**: 모바일, 태블릿, 데스크톱 모든 기기에서 최적화된 화면
- **🖼️ 썸네일 그리드**: 영상을 썸네일 카드 형태로 시각적으로 표시
- **🔍 실시간 검색**: 영상 제목으로 빠른 검색 및 필터링
- **🎬 새창 열기**: 썸네일 클릭 시 Vimeo 상세 페이지를 새창으로 열기
- **🚀 일괄 작업**: 모든 영상을 한 번에 순차적으로 열기
- **📊 실시간 통계**: 총 영상 수, 열린 영상 수, 검색 결과 수 표시
- **⚡ 자동 로딩**: 페이지 로드 시 자동으로 영상 목록 불러오기

## 🚀 사용 방법

1. **파일 실행**: `vimeo_viewer.html` 파일을 웹 브라우저에서 열기
2. **자동 로드**: 페이지가 열리면 자동으로 Vimeo 영상 목록을 불러옵니다
3. **영상 탐색**: 썸네일 카드를 클릭하여 Vimeo에서 영상 시청
4. **검색 사용**: 상단 검색창에서 영상 제목으로 필터링

## 🎮 버튼 기능

### 상단 컨트롤 버튼
- **📺 영상 목록 새로고침**: Vimeo에서 최신 영상 목록 다시 불러오기
- **🚀 모든 영상 열기**: 현재 표시된 모든 영상을 순차적으로 새창으로 열기
- **ℹ️ 정보**: 애플리케이션 정보 및 사용법 안내

### 영상 카드 기능
- **썸네일 클릭**: 해당 영상의 Vimeo 상세 페이지를 새창으로 열기
- **호버 효과**: 마우스 오버 시 썸네일 확대 및 재생 버튼 표시
- **영상 정보**: 제목, 재생 시간, 조회수 표시

## 🔍 검색 기능

- **실시간 검색**: 검색창에 입력하면 즉시 필터링
- **제목 검색**: 영상 제목의 일부만 입력해도 검색 가능
- **Enter 키**: 검색창에서 Enter 키로 검색 실행
- **검색 초기화**: 검색창을 비우면 모든 영상 다시 표시

## 🎯 Vimeo API 연동

### API 엔드포인트
```
https://vimeo.com/api/v2/user/104908181/videos.json
```

### 제공 정보
- 영상 ID 및 URL
- 썸네일 이미지 (고해상도)
- 영상 제목
- 재생 시간
- 조회수
- 업로드 날짜

### 오류 처리
- API 연결 실패 시 샘플 데이터로 데모 표시
- 썸네일 로드 실패 시 플레이스홀더 이미지 표시
- 네트워크 오류 시 사용자 친화적 오류 메시지

## 🔧 기술 사양

- **HTML5**: 시맨틱 마크업
- **CSS3**: Flexbox, Grid, 애니메이션, 반응형 디자인
- **JavaScript**: ES6+ 문법, Fetch API, 모던 브라우저 지원
- **Vimeo API**: RESTful API를 통한 영상 데이터 가져오기

## 📊 통계 정보

하단에 실시간으로 표시되는 통계:
- **총 영상 수**: Vimeo에서 불러온 전체 영상 개수
- **열린 영상**: 현재까지 새창으로 연 영상 개수
- **검색 결과**: 현재 검색 필터에 맞는 영상 개수

## ⚠️ 주의사항

1. **API 제한**: Vimeo API 정책에 따라 일부 영상이 표시되지 않을 수 있습니다
2. **네트워크**: 안정적인 인터넷 연결이 필요합니다
3. **CORS 정책**: 일부 브라우저에서 CORS 정책으로 인해 API 호출이 제한될 수 있습니다
4. **로그인 필요**: 일부 영상은 Vimeo 계정 로그인이 필요할 수 있습니다
5. **새창 차단**: 브라우저의 팝업 차단 기능을 해제해야 새창 열기가 가능합니다

## 💡 사용 팁

2. **검색 활용**: 많은 영상이 있을 때는 검색 기능으로 빠르게 찾기
3. **일괄 열기**: 여러 영상을 한 번에 보고 싶을 때는 "모든 영상 열기" 사용
4. **새로고침**: 새로운 영상이 업로드되었을 때는 새로고침 버튼 사용

## 🛠️ 문제 해결

### 영상이 로드되지 않는 경우
1. 네트워크 연결 확인
2. 브라우저 새로고침 (F5)
3. "영상 목록 새로고침" 버튼 클릭
4. 브라우저 개발자 도구에서 오류 메시지 확인

### 새창이 열리지 않는 경우
1. 브라우저 팝업 차단 설정 확인
2. 해당 사이트를 팝업 허용 목록에 추가
3. 브라우저 설정에서 팝업 허용
